# PSU Sphere

 PSU Sphere is a centralized web application designed to manage student organizations, programs, and memberships within the university. It provides a structured platform for handling student records and organizational memberships in one place.



# PSU Sphere Features

**College Management** 

- Add, edit, and delete college records.

- Organize programs and organizations under specific colleges.

**Program Management**

- Create and manage academic programs.

- Associate programs with their respective colleges.

**Student Records Management**

- Register student information including ID, name, and program.

- View, update, and manage student data.

**Organization Management**

- Create and manage student organizations.

- Assign organizations to specific colleges.

- Store organization descriptions and details.

**Membership Tracking**

- Assign students to organizations.

- Record the date a student joined an organization.

- Manage organization membership records.

**Admin Dashboard**

- Centralized interface for managing all data.

- Search, filter, and organize records efficiently.

**Automated Data Generation (Optional for testing)**
- Use the Faker-based command to generate sample organizations, students, and memberships.
